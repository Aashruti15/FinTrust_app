# FinTrust_app
Interactive platform analyzing ATM transactions, visualizing trends, predicting cash demand.


# Overview

FinTrust ATM Intelligence Platform is an interactive analytics platform that is created to visualize and analyze the behavior of ATM transactions using the data visualization and exploratory data analysis (EDA) techniques. The platform resembles an actual banking analytics scenario where users get to learn the pattern of withdrawal, operational demand, and trends of transactions using an interactive dashboard that is visually appealing.

The application is a combination of data exploration, visualization, and predictive simulation to show how an efficient way of managing cash in ATM may be achieved through data-driven decision-making.

 
# Objective
The primary objective of this application is to:

Analyze ATM transaction and withdrawal patterns using data analytics.

Perform exploratory data analysis to identify behavioural trends.

Visualize financial activity through interactive dashboards.

Simulate cash demand prediction based on user inputs.

Demonstrate how analytics supports operational decision-making in banking systems.

 
# Target Audience
This platform is designed for:

Banking operations teams monitoring ATM performance

Data analysts exploring financial datasets

Students learning data mining and analytics concepts

Business managers interested in data-driven insights

Educators demonstrating applied analytics systems
 
 
# User Persona (User Problems)
Persona: Operations Analyst – Banking Sector
Background

Works in ATM operations or financial analytics.

Responsible for ensuring ATMs remain functional and well-stocked.

Problems Faced

ATMs running out of cash unexpectedly.

Difficulty identifying demand patterns across locations.

Large datasets that are hard to interpret quickly.

Lack of interactive tools for real-time exploration.

Inefficient manual decision-making processes.

Needs
Clear visual insights.

Easy data exploration tools.

Predictive understanding of cash demand.

Simple yet powerful analytics interface.

 
# User Research Insights & Survey Summary

Key Insights Identified
User research and simulated survey findings highlighted several common needs:

Users prefer visual dashboards over raw spreadsheets.

Interactive filtering improves understanding of data trends.

Simple prediction tools increase decision confidence.

Dark/light theme options improve usability and accessibility.

Structured navigation enhances engagement and usability.
